
# Concept - 1

## 🔹 Problem (in simple words)

You are given a number **d**.

You must **construct a number (as a string of digits)** such that:

1. The number is divisible by **d**
2. The **sum of its digits** is also divisible by **d**
3. The length of the number must be **≤ 4000**
4. You just need to **print any valid number**


### 🔸 Idea

Just **repeat the number `d` exactly `d` times** and concatenate it.

### 🔸 Example

If

```
d = 12
```

Construct:

```
12121212121212121212   (12 repeated 12 times)
```
